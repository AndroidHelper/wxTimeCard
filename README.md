# wxTimeCard
###### 1、微信小程序，次数会员卡商城，购买到的次数会员卡，有充值、付款码、订单交易记录、电子卡页面
###### 2、对公共的模块进行组件封装，已达到复用目的
###### 3、页面采用flex弹性布局

# 注意事项
1、需要自行修改project.config.json文件中的"appid"值。
